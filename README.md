# Water Billing System 💧

## 📌 Description

A web application developed with Python and Flask for automated water billing, including data storage and bill generation.


## 🛠️ Technologies Used

* Python
* Flask
* SQLite3
* HTML, CSS

## 🚀 Features

* Calculate water consumption
* Generate monthly bills
* Store customer data
* Simple user interface

## ▶️ How to Run

1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
